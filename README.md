# Local-Fuel-Price-Tracker
This is a personal project using the Gov.UK One Fuel Finder API

My use-case for this would be to create a small dedicated device to place in my car using an ESP or Arduino microcontroller 

I intend to create a Python program to pull the current fuel prices from this API and present the user with several options based on price and distance

To limit data processing I imagine I will need to limit it to an area possibly using current location from a gps peripheral

All information will be presented on a small OLED but ideally a mini-map style display would be cool

I intend to keep this repository public but will not necissarily have the most up to date content uploaded

I also need to check security term for the API usage to see if there are any restrictions
